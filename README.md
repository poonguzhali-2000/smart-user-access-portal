# Smart User Access Portal
A role-based Single Page Application built using Angular, Spring Boot, and MariaDB.

---

# Tech Stack

## Frontend
- Angular
- PrimeNG
- Reactive Forms
- Angular Guards
- Angular Interceptors

## Backend
- Spring Boot
- Spring Security
- Spring Data JPA
- Maven
- JWT Security
- BCrypt Password Encryption

## Database
- MariaDB

---

# Features
- User Login
- Role-Based Access
- Admin User Management
- User Records Display
- REST API Integration
- Async API Delay Simulation

# Run Application

## Backend

```bash
cd backend
mvn spring-boot:run
```

Backend runs on:
http://localhost:8080

---

## Frontend

```bash
cd angular
ng serve
```

Frontend runs on:
http://localhost:4200
