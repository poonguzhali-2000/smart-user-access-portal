## Role Enum

Created Role enum to manage fixed user roles.

Why Enum?
- Prevents typo mistakes
- Improves type safety
- Professional backend practice

Used:
@Enumerated(EnumType.STRING)

This stores readable values like:
- ADMIN
- GENERAL_USER

instead of:
- 0
- 1

## BCrypt Password Encryption

Implemented BCryptPasswordEncoder for secure password storage.

Passwords are encrypted before saving into database.

Why BCrypt?
- Industry standard password hashing
- Prevents plain-text password storage
- Improves backend security

Created:
- PasswordConfig class - removed
- Configured PasswordEncoder bean inside SecurityConfig.

Updated:
- AppUserService saveUser() method

## Frontend Authentication Flow

Implemented login authentication flow between Angular and Spring Boot.

Flow:
1. User enters credentials
2. Angular Reactive Form validates input
3. AuthService calls backend API
4. Spring Boot validates credentials
5. JWT token returned
6. Token stored in localStorage
7. User redirected to dashboard
