## JPA Entity

---

## Role Enum

Created Role enum to manage fixed user roles.

Why Enum?
- Prevents typo mistakes
- Improves type safety
- Professional backend practice

Used:
@Enumerated(EnumType.STRING)

This stores readable values like:
- ADMIN
- GENERAL_USER

instead of:
- 0
- 1

## BCrypt Password Encryption

Implemented BCryptPasswordEncoder for secure password storage.

Passwords are encrypted before saving into database.

Why BCrypt?
- Industry standard password hashing
- Prevents plain-text password storage
- Improves backend security

Created:
- PasswordConfig class

Updated:
- AppUserService saveUser() method
